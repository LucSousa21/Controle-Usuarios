Este é um projeto pessoal desenvolvido para praticar conceitos fundamentais de Estruturas de Dados usando a linguagem C.

  -O sistema implementa uma lista simplesmente ligada, permitindo:
  -Inserção de usuários no início da lista
  -Remoção de usuários no início
  -Listagem completa dos usuários cadastrados
  -Navegação por meio de um menu interativo

Este projeto foi criado para praticar:

  -Manipulação de ponteiros e ponteiros para ponteiros
  -Alocação dinâmica com malloc e free
  -Estruturação de uma lista encadeada
  -Implementação de nós contendo múltiplos campos
  -Divisão de código em arquivos (.h e .c)
  -Criação de um menu simples para interação com o usuário

Ao executar o programa, o usuário encontra um menu como este:

  1 - Cadastrar usuário
  2 - Listar usuários
  3 - Remover usuário
  4 - Sair
